# Criando Uma Calculadora Com React

**Descrição:**

Uma calculadora simples construída com React, permitindo operações básicas como adição, subtração, multiplicação e divisão. A aplicação utiliza componentes funcionais e hooks para gerenciar o estado e as interações do usuário.

**Funcionalidades:**

* Realiza operações aritméticas básicas
* Limpa o display
* Permite inserir números e operadores
* Calcula o resultado final

**Tecnologias Utilizadas:**

* React
* JavaScript

**Estrutura do Projeto:**

* **components:** Contém os componentes reutilizáveis da aplicação, como Display, Botões e Teclado.
* **styled-components:** Consiste em uma biblioteca para React e React Native que permite escrever CSS dentro do JavaScript.
* **App.js:** Componente principal que renderiza a calculadora.
* **hooks:** Permite usar recursos como estado e efeitos colaterais em componentes funcionais, sem a necessidade de classes.
* **useState:** Adicionar estado local a componentes funcionais, permitindo que eles mantenham e atualizem dados ao longo do tempo.
